# SIGA – Sistema de Acompanhamento de Entregas

`CURSO: ANÁLISE E DESENVOLVIMENTO DE SISTEMAS`

`DISCIPLINA: Projeto Semestral`

`SEMESTRE: 01/2026`

O SIGA é um aplicativo desenvolvido para facilitar o acompanhamento do status de entregas de forma simples, organizada e transparente. A solução permite que empresas registrem e gerenciem encomendas, que entregadores atualizem o andamento das entregas e que clientes acompanhem o histórico e o status de seus pedidos por meio de um código de rastreamento.

O sistema busca reduzir a incerteza durante o processo de entrega, melhorar a comunicação entre clientes, empresas e operadores logísticos e oferecer maior rastreabilidade das movimentações da encomenda. A aplicação será desenvolvida com arquitetura distribuída, utilizando backend em .NET, frontend em React/React Native e persistência de dados em banco NoSQL.

O projeto tem como objetivo apoiar a gestão de entregas, aumentar a transparência das informações e melhorar a experiência do usuário durante o processo de recebimento de pedidos.
