# Contexto do Projeto

O SIGA é um sistema acadêmico de gestão e rastreamento de entregas.

Esta versão pública está sendo preparada para portfólio, com foco em apresentar a arquitetura, as funcionalidades implementadas e a evolução técnica do projeto.

A documentação definitiva será reescrita após a limpeza do repositório, considerando:

- backend com APIs em .NET;
- frontend web em React;
- aplicativo mobile com Expo/React Native;
- MongoDB próprio;
- deploy próprio;
- remoção de módulos legados não utilizados.
